1 #font-awesome Font Awesome

Font Awesome gives you scalable vector icons that can instantly be customized, size, color, drop shadow, and anything that can be done with the power of CSS.

1.1 #font-awesome.1 Icons

You can place Font Awesome icons just about anywhere using the CSS Prefix fa and the icon's name. Font Awesome is designed to be used with inline elements (we like the <i> tag for brevity, but using a <span> is more semantically correct).

For a full list of available icons, visit the Font Awesome documentation.

Examples
Default styling
.fa-camera-retro
Camera Retro
.fa-thumbs-o-up
Thumbs Up
.fa-cogs
Cogs 
<i class="fa [modifier class]"></i>

1.3 #font-awesome.3 Fixed Width Icons

Use fa-fw to set icons at a fixed width. Great to use when different icon widths throw off alignment. Especially useful in things like nav lists & list groups.

Example
  Home   Library   Applications   Settings 
<div class="list-group">
  <a class="list-group-item" href="#"><i class="fa fa-home fa-fw"></i>&nbsp; Home</a>
  <a class="list-group-item" href="#"><i class="fa fa-book fa-fw"></i>&nbsp; Library</a>
  <a class="list-group-item" href="#"><i class="fa fa-pencil fa-fw"></i>&nbsp; Applications</a>
  <a class="list-group-item" href="#"><i class="fa fa-cog fa-fw"></i>&nbsp; Settings</a>
</div>

1.6 #font-awesome.6 Animated Icons

1.8 #font-awesome.8 Stacked Icons

To stack multiple icons, use the fa-stack class on the parent, the fa-stack-1x for the regularly sized icon, and fa-stack-2x for the larger icon. fa-inverse can be used as an alternative icon color. You can even throw larger icon classes on the parent to get further control of sizing.

Example
fa-twitter on fa-square-o
fa-flag on fa-circle
fa-terminal on fa-square
fa-ban on fa-camera 
<span class="fa-stack fa-lg">
  <i class="fa fa-square-o fa-stack-2x"></i>
  <i class="fa fa-twitter fa-stack-1x"></i>
